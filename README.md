# ahmetrende/homebrew-tap

Homebrew tap for my macOS apps.

## BadgeBar

Mirror app unread badges (Mail, Slack, Teams, …) into the macOS menu bar.

```sh
brew install --cask ahmetrende/tap/badgebar
```

After install, see the on-screen caveats: BadgeBar isn't notarized yet, so the
first launch needs **right-click → Open** (or *System Settings → Privacy &
Security → Open Anyway*), then grant **Accessibility** permission.

Source & releases: https://github.com/ahmetrende/BadgeBar
